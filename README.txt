# Investment Calculator App

This is a simple **Streamlit web app** that calculates the future value of your investments based on:
- Initial investment
- Monthly contributions
- Annual return
- Annual increase in contributions
- Bank fees

## Try it Online

Deploy it instantly via [Streamlit Cloud](https://streamlit.io/cloud).
